# Wireless Concepts

- Productivity is no longer restricted to a fixed location.
- Users can now roam freely and wirelessly.

## Concerns

- **Coverage Area:** Where to place access points.
- **Interference:** Public areas or home applications.
- **Security:** Recipients can read traffic.

## Wireless IEEE Standards

- IEEE 802.11 standard.
- Utilizes CSMA/CA.

**Variation Cases:** 802.11a, b, g, n, ac and ad.
**Bluetooth:** 802.15.
**WiMAX:** 802.16.

## Wireless Access Point (WAP)

- Provides radio interface to network.

## Wireless Router

- Provides access point and Ethernet switching plus routing capabilities.

## Wireless NICs

- Has a wireless NIC, infrastructure device.
- i.e., router or WAP.

## Wireless Home Solutions

- Small office / home office.
- Home users interconnect wireless devices with their own integrated wireless router.
- Serves as: Access point, Ethernet switch, Router.

- Each access point (WAP) is configured to manually and individually.
- Interference might happen when they are too close together.
- A VLAN can configure a wireless network to be controller based.
