# Wireless Security

## Evil Twin Access Point Attack

- Man-in-the-middle type attack where a rogue access point configures to the same SSID as a legitimate access point (AP).

- Used in public areas with WiFi.
- Sometimes unnoticed and return traffic from legitimate access points is sent to the rogue access point (AP).
